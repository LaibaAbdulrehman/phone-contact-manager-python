
# Phone Contact Manager - Python Project

A simple yet practical **menu-driven Phone Contact Book** built in Python using dictionaries (hash maps) to store and manage contacts efficiently.

![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python&logoColor=white)
![Beginner Friendly](https://img.shields.io/badge/Level-Beginner-green)

## Features
- Add a new contact (name + phone number)
- Search for a contact by name
- Delete a contact
- View all saved contacts
- User-friendly command-line menu interface (CLI)

## Technologies & Concepts Used
- **Python** (core programming language)
- **Dictionaries** – for fast lookups, insertions, and deletions
- Functions for modular and reusable code
- Basic input/output handling
- Loops, conditionals, and infinite loop with break for menu control
- Clean, readable, and well-structured code

## What I Learned
- Real-world application of Python dictionaries (key-value mapping)
- Breaking code into functions for better organization and reusability
- Handling user input in a loop-based interactive program
- Building a complete menu-driven CLI application from scratch
- Importance of writing clear and maintainable code

## How to Run
1. Clone or download this repository:
   ```bash
   git clone https://github.com/LaibaAbdulrehman/python-contact-manager.git
   ```
2. Navigate to the project folder:
   ```bash
   cd python-contact-manager
   ```
3. Run the program:
   ```bash
   python contact_management.py
   ```
